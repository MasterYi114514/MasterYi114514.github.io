变量

变量就是编程里用来存放数据的盒子。

——豆包

  1. 变量有名字（变量名，是一个标识符），可以用于找到它
  2. 变量里能存储数据（数字、文字、真假值等）
  3. 变量里的数据可以修改

常见的变量类型

整型变量

修饰符signed和unsigend

  • 默认为有符号(signed)，表面变量可以表示正数、负数和0。
  • 用unsigned修饰后，变量只能表示非负整数，即0和正数

常见整型变量空间占用：

数据类型 32位系统（x86） 64位系统（x64）
int 4字节（32位） 8字节（64位）
short/short int 2字节（16位） 2字节（16位）
long/long int 4字节（32位） 8字节（64位）
long long/long long int 8字节（64位） 8字节（64位）
unsigned 类型 和对应的有符号类型相同 和对应的有符号类型相同

浮点型变量

  • float：单精度浮点型，占用4个字节（32位），大约能提供7位有效数字。
  • double：双精度浮点型，占用8个字节（64位），大约能提供15-17位有效数字，精度比float更高。
  • long double：长双精度浮点型，占用的字节数在不同系统中可能不同，通常比 double 提供更高的精度。

变量的初始化方式

1. 默认初始化：变量不写任何初值，由编译器自动处理。

int global_var;         // 未初始化全局变量被自动置 0

int main()
{
    int a;         // 局部内置变量，垃圾值
    std::string str;       // 类：调用无参构造函数
    int* p = new int;      // 堆内存默认初始化，垃圾值
}

2. 空值初始化：用空括号（()或{}）,强制清零/调用默认构造

int a{};
int b();
int* p = new int();
int arr[5]{ };

3. 列表初始化：用花括号{}，统一语法，禁止窄化转换

int a{10};
int b = {20};
int arr[]{1, 2, 3};
std::vector<int> v{1, 2, 3, 4};

1) 禁止窄化转换：

int a = 3.14;      // 允许阶段
int b{3.14};       // 编译报错

2) 支持std::initializer_list多参数批量构造：

struct Vec
{
    Vec(std::initializer_list<int> list);
}

4. 直接初始化：用圆括号()，直接调用构造函数

int a(100);
struct Point { int x, y; }
Point p(1, 2);

5. 拷贝初始化：用=赋值，触发拷贝构造/隐式转换

int a = 10;
std::string s = "Hello";
class A;
A a = A();

6. 聚合初始化：针对简单的结构体、数组，用{}批量赋值

struct Position { double x, y; }
Position p = { 2.5, 3.4 };
int arr[4] = { 1, 2 };

常量

常量（constant） 是指在程序运行过程中其值不能改变的量。

在C++中，使用const关键字来定义常量，语法为const 数据类型 常量名 = 常量值;。

const double pi = 3.14159;

定义常量的细节

1. 常量必须在定义时初始化。

#include <iostream>

int main()
{
    const int a;
    return 0;
}

编译后，得到以下报错：

code.cpp: In function 'int main()':
code.cpp:5:15: error: uninitialized 'const a' [-fpermissive]
    5 |     const int a;
      |               ^

2. 常量的值在程序运行过程中不能被修改，如果尝试修改常量的值，编译器会报错。

#include <iostream>

int main()
{
    const int b = 10;
    b = 20;

    return 0;
}

编译后，得到以下报错：

code.cpp: In function 'int main()':
code.cpp:6:7: error: assignment of read-only variable 'b'
    6 |     b = 20;
      |     ~~^~~~

字面常量

字面常量

字面常量是在代码中直接书写的固定值，其类型由书写形式决定。

  1. 整数常量：如10、-20、30000等，默认是int类型，如果值过大，可能会是 long 或 long long 类型。
  2. 浮点常量：如3.14、2.71828等，默认是double类型。
  3. 字符常量：用单引号括起来的单个字符，如 'a'、'A'、'0' 等。
  4. 字符串常量：用双引号括起来的零个或多个字符序列，如 "Hello"、"what the hell" 等

字面量后缀

可以用后缀来明确字面量的类型。

1. 整数字面量后缀

  • u或U：表示无符号整数（unsigned）。
  • l 或 L：表示长整数（long）。
  • ll 或 LL：表示长长整数（long long）。
unsigned int num1 = 10u;
unsigned long num2 = 100UL;
long long big_num = 1000000000000000000000LL;

2. 浮点字面量后缀

  • f或F：表示单精度浮点数（float）。
  • l 或 L：表示长双精度浮点数（long double）。
float pi = 3.14f;
long double e = 2.71828182845904523536L;

编译时常量/常量表达式（constant expression）

注：constexpr关键字在 C++11 中才引入。

编译时常量和普通常量一样，也必须在定义时初始化，并且在程序运行的过程中不能被修改。

在C++中，使用constexpr关键字来定义编译时常量：

// after C++11
#include <iostream>

int main()
{
    constexpr int a = 10;

    return 0;
}

编译时常量的值在编译阶段就能确定，随后就会被直接替换到代码中，这样在程序运行时就不需要再查找地址、访问内存来获取值了，因此编译时常量的访问效率更高。

编译时就能确定的值：只由字面量和其它编译时能确定的值组成。

const int x = 10;      // 10是字面量，因此x可以在编译时求出结果
const int y = 20  *x;  // x可在编译时求出结果，故y也可在编译时求出结果

普通的常量既可以在编译时就确定值，也可以在运行的时候才确定值，前者可以被用来初始化编译时常量，后者则不能。

#include <iostream>

int main()
{
    int variable = 10;              // variable 是一个变量
    const int a = variable;         // a 是一个常量，值在运行时才确定
    constexpr int b = variable;     // 错误，编译时常量不能被变量初始化
    constexpr int c = a;            // 错误，编译时常量不能被运行时才确定的常量初始化

    return 0;
}
