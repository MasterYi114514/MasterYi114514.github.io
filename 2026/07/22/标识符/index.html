标识符

发表于2026-07-22|更新于2026-07-25|Cpp
|浏览量:

标识符（identifier）：在程序中用来充当变量、函数、类等的名字。

标识符的命名规则

1.标识符只能由字母、数字和下划线组成，并且必须以字母或下划线开头。

int myVariable = 10;        // 正确，myVariable 以字母开头
int _myVariable = 20;       // 正确，_myVariable 以下划线开头
int myVariable2 = 40;       // 正确，可以包含数字
int 2myVariable = 30;       // 错误，可以包含数字但不能以数字开头
int my-Variable = 40;       // 错误，标识符中不能包含其它字符

2.标识符不能是 C++ 的关键字。

这里 C++ 的关键字不需要特殊记忆，因为在编程过程中会经常接触到它们，自然而然就会记住它们了。

int int = 10;              // 错误，int 是 C++ 的关键字
int while = 20;            // 错误，while 是 C++ 的关键字
int return = 30;           // 错误，return 是 C++ 的关键字

3.标识符严格区分大小写，即字母相同而大小写不同则表示不同的标识符。

int a = 10;              // a 是一个标识符
int A = 20;              // A 是另一个标识符，与 a 不同

int my_variable = 30;   // m、v 都小写
int My_variable = 40;   // m 大写，v 小写
int my_Variable = 50;   // m 小写，V 大写
int My_Variable = 60;   // m、V 都大写

常见的标识符命名法

小驼峰命名法

小驼峰命名法（lower camel case）：第一个单词的首字母小写，后续单词的首字母大写，且不使用下划线分隔单词。

int myVariable = 10;        // 使用小驼峰命名法命名变量
void getUserInfo();         // 使用小驼峰命名法命名函数

大驼峰命名法

大驼峰命名法（upper camel case）：每个单词的首字母都大写，且不使用下划线分隔单词。

class StudentManager;       // 使用大驼峰命名法命名类
struct EmployeeRecord;      // 使用大驼峰命名法命名结构体
enum ColorType;             // 使用大驼峰命名法命名枚举

下划线命名法（蛇形命名法）

下划线命名法（snake case）：所有字母小写，单词之间使用下划线分隔。

int it_is_a_variable = 10;     // 使用下划线命名法命名变量
void get_user_info();          // 使用下划线命名法命名函数
class student_manager;         // 使用下划线命名法命名类
struct employee_record;        // 使用下划线命名法命名结构体
enum color_type;              // 使用下划线命名法命名枚举
