lambda 表达式

通过使用lambda表达式可以在代码中就地定义短小的函数逻辑，同时能捕获外层作用域的变量作为内部状态，其核心语法如下：

[捕获列表](参数列表)
{
    函数体
}

例如：

[]() { std::cout << "this is lambda!" << std::endl; }

1. 无参数时，括号可以省略

[] { std::cout << "this is lambda!" << std::endl; }

2. 可以在函数体后紧跟一个括号，表示直接调用该匿名函数

[] { std::cout << "this is lambda!" << std::endl; }();

lambda 表达式的捕获方式

值捕获

值捕获：捕获变量的副本，在lambda定义时完成拷贝。

  • 显式捕获：写法为[变量名1, 变量名2, ...]，用于显式捕获外界的变量
  • 隐式捕获：写法为[=]，直接捕获所有外层的可用变量
int a = 10, b = 100;
auto f1 = [a]() { return a; };
auto f2 = [=]() { return a + b; };

a = 20;
std::cout << f1(); << std::endl;     // 输出10
std::cout << f2();  << std::endl;   // 输出 110

引用捕获

引用捕获：捕获变量的引用

  • 显式捕获：写法为[&变量名1, &变量名2, ...]，用于显式捕获外界的变量
  • 隐式捕获：写法为[&]，用于隐式捕获外界所有可用变量的引用
int a = 10, b = 100;
auto f1 = [&a]() { return a; };
auto f2 = [&]() { return a + b; };

a = 20;
std::cout << f1(); << std::endl;     // 输出20
std::cout << f2();  << std::endl;    // 输出 120

混合捕获

混合捕获：组合 隐式全捕获 + 部分变量例外

  • [=, &变量1, &变量2, ...]：除变量1、变量2等单独列出的变量进行引用捕获，其余变量均进行值捕获。
  • [&, 变量1, 变量2, ...]：除变量1、变量2等单独列出的变量进行值捕获，其余变量均进行引用捕获。
  • 错误写法：[=, &]，不允许同时指定两种隐式规则。
int x, y, z, w;
x = y = z = w = 10;

auto f1 = [=, &z, &w] { return x + y + z + w; };
auto f2 = [&, z, w] { return x + y + z + w; };

x += 1, y += 2, z += 3, w += 4;
std::cout << f1(); << std::endl;
std::cout << f2(); << std::endl;
